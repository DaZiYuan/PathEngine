# 使用路径字符串轻松读写Windows资源
 - [x] 真实路径解析
   - [x] 环境变量解析
      - 例：`path:\%appdata%\path`
   - [ ] 自定义路径解析
      - 例：`path:\%xx%\path`
   - [x] 模糊匹配文件
      - 例：`path:\%xx%\path\*\*\*.txt`
   - [x] 模糊匹配文件夹
      - 例：`path:\%xx%\path\*\*\`，注意斜线结尾
    
 - [ ] 内容获取
   - [x] 获取获取注册表内容

        - 例：`version:\path`
   - [x] 获取文件内容
        - 例：`content:\path`
   - [x] 获取Dll嵌入资源内容
        - 例：`embedded:\path`
